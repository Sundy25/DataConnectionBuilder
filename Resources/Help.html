Data Connection Builder Help

What is Data Connection Builder?

Data Connection Builder is used to build connection strings for use with Microsoft's ActiveX Data Objects .NET (ADO.NET) Framework Data Provider for Object Linking and Embedding, Database (OLE DB) technology. Such a connection string contains the information needed to establish a connection to an OLE DB data source through ADO.NET. This information typically includes, at minimum, the name of the data source and the provider.

Data Connection Builder supports the three Microsoft OLE DB providers that have been tested with ADO.NET: SQL Server, Oracle, and ACE (used with Access and Excel). A connection string might also include security information, such as user ID and password. For more information, see Connection String Syntax (OleDb Connection Strings). ConnectionStrings.com also provides information on connection strings for many additional OLE DB providers.


Building a new connection string

  1. Specify a Provider and a Data Source. These properties are always required.
  2. Specify other properties as appropriate. Certain properties may be required depending on the provider. Properties that do not apply to the specified provider are hidden.
  3. It is always a good idea to make sure that the connection string is valid once it is built. Test the connection string by clicking the Test Connection button.


Troubleshooting

If a data provider is not installed, an error will occur when the connection is tested. Below are the errors that typically occur and links for downloading each data provider.

Provider Error Link(s)
ACE (Access/Excel) The 'Microsoft.ACE.OLEDB.12.0' provider is not registered on the local machine.
  • 2007 Office System Driver: Data Connectivity Components
  • Microsoft Access Database Engine 2010 Redistributable
Oracle The 'MSDAORA.1' provider is not registered on the local machine.
  • Oracle Instant Client
SQL Server [DBNETLIB][ConnectionOpen (Connect()).]SQL Server does not exist or access denied.
  • SQL Server Native Client
